Bisection Method

Definition

The Bisection Method is a root-finding technique that repeatedly divides an interval in half and then selects the subinterval in which a root exists. It is a type of bracketing method and is based on the Intermediate Value Theorem.

Requirements

  • A continuous function f(x).
  • An interval [a, b] such that f(a) and f(b) have opposite signs, i.e. f(a) * f(b) < 0.

Procedure

  1. Start with a continuous function f(x) on the interval [a, b] where f(a) * f(b) < 0.
  2. Compute the midpoint c = (a + b) / 2.
  3. Evaluate f(c):
    • If f(c) = 0, then c is the root.
    • If f(c) * f(a) < 0, then set b = c.
    • Else, set a = c.
  4. Repeat the above steps until the interval [a, b] is sufficiently small or a desired precision is achieved.

Example JavaScript Implementation


function bisection(f, a, b, tol = 1e-6, maxIter = 100) {
  if (f(a) * f(b) >= 0) {
    throw new Error("Function has the same signs at a and b");
  }
  let mid;
  for (let i = 0; i < maxIter; i++) {
    mid = (a + b) / 2;
    const fMid = f(mid);
    if (Math.abs(fMid) < tol || (b - a) / 2 < tol) {
      return mid;
    }
    if (fMid * f(a) < 0) {
      b = mid;
    } else {
      a = mid;
    }
  }
  return mid;
}

// Example usage:
const root = bisection(x => x ** 3 - x - 2, 1, 2);
console.log("Root:", root);

Advantages

  • Simple and easy to implement.
  • Always converges (if the initial interval satisfies the condition).
  • Guaranteed to find a root in the interval.

Disadvantages

  • Slow convergence compared to other methods (e.g., Newton-Raphson).
  • Requires the function to be continuous and for the root to be bracketed.
