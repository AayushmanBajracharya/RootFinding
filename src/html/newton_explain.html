Newton-Raphson Methods

Newton-Raphson Method - Definition

The Newton-Raphson Method is a powerful open root-finding method that uses tangents to approximate the root of a function. It requires the function and its derivative.

Newton-Raphson Method - Requirements

  • A differentiable function f(x).
  • Its derivative f'(x).
  • An initial guess x0.

Newton-Raphson Method - Procedure

  1. Start with an initial guess x0.
  2. Compute x1 = x0 - f(x0) / f'(x0).
  3. Repeat using x0 = x1 until the difference between consecutive values is less than a tolerance level.

Newton-Raphson Method - Example JavaScript Implementation


function newtonRaphson(f, df, x0, tol = 1e-6, maxIter = 100) {
  for (let i = 0; i < maxIter; i++) {
    const fx = f(x0);
    const dfx = df(x0);
    if (Math.abs(dfx) < Number.EPSILON) {
      throw new Error("Derivative near zero");
    }
    const x1 = x0 - fx / dfx;
    if (Math.abs(x1 - x0) < tol) {
      return x1;
    }
    x0 = x1;
  }
  return x0;
}

// Example usage:
const root = newtonRaphson(x => x ** 3 - x - 2, x => 3 * x ** 2 - 1, 1.5);
console.log("Root:", root);
