Secant Methods

Secant Method - Definition

The Secant Method is an open root-finding method that uses two initial points to generate a sequence approximating the root of a function. It uses a secant line through the points to estimate the root.

Secant Method - Requirements

  • A continuous function f(x).
  • Two initial guesses x0 and x1.

Secant Method - Procedure

  1. Start with initial values x0 and x1.
  2. Compute f(x0) and f(x1).
  3. Use the formula: x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0)).
  4. Update x0 = x1 and x1 = x2.
  5. Repeat until |x1 - x0| is smaller than a tolerance level or maximum iterations are reached.

Secant Method - Example JavaScript Implementation


function secant(f, x0, x1, tol = 1e-6, maxIter = 100) {
  for (let i = 0; i < maxIter; i++) {
    const f0 = f(x0);
    const f1 = f(x1);
    if (Math.abs(f1 - f0) < Number.EPSILON) {
      throw new Error("Division by near-zero");
    }
    const x2 = x1 - f1 * (x1 - x0) / (f1 - f0);
    if (Math.abs(x2 - x1) < tol) {
      return x2;
    }
    x0 = x1;
    x1 = x2;
  }
  return x1;
}

// Example usage:
const root = secant(x => x ** 3 - x - 2, 1, 2);
console.log("Root:", root);

Advantages of Secant Method

  • Does not require the function to change signs.
  • Faster convergence than bisection in many cases.

Disadvantages of Secant Method

  • Not guaranteed to converge.
  • Can fail if denominator approaches zero.
